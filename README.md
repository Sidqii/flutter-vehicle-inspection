# Vehicle Inspection App

Flutter technical assessment project for a vehicle inspection form used by leasing field surveyors.

## Features

- Vehicle inspection form
- Indonesian license plate validation
- Kilometer validation
- Vehicle condition selection
- GPS location capture
- Vehicle photo documentation
- Digital signature pad
- Autosave draft
- Offline data storage support

## Tech Stack

- Flutter
- Dart
- Geolocator
- Image Picker
- Shared Preferences

## Inspection Data

The form collects:

- License plate number
- Vehicle photos: front, rear, left, right, and speedometer
- Exterior condition
- Engine condition
- Current kilometer
- GPS location
- Address note
- Vehicle movement status
- Additional notes
- Digital signature

## Notes

This project focuses on the main inspection flow, form validation, GPS capture, photo documentation, digital signature, autosave, and offline draft storage.

Advanced features such as photo watermarking, image compression, background sync, fake GPS detection, and face verification can be developed as future improvements.
